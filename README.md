<h1>LKS-Kisi-Kisi-dan-Soal</h1>
Repositori ini merupakan kumpulan arsip Kisi-kisi dan Soal Lomba Kompetensi Siswa (LKS) SMK dari berbagai tahun. Koleksi ini mencakup tingkat Kabupaten, Provinsi, hingga Nasional untuk membantu siswa dan pembimbing dalam mempersiapkan kompetisi teknis.

<h3>📌 Daftar Bidang Lomba</h3>
Arsip ini mencakup materi untuk bidang-bidang berikut:
<ul>
  <li>IT Network System Administration (ITNSA)</li>
  <li>Cloud Computing</li>
  <li>Web Technologies</li>  
  <li>IT Software Solution for Business (Aplikasi Perkantoran/Bisnis)</li>
  <li>Graphic Design Technology</li>
</ul>
