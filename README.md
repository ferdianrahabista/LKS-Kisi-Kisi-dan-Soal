LKS-Kisi-Kisi-dan-Soal
Repositori ini merupakan kumpulan arsip Kisi-kisi dan Soal Lomba Kompetensi Siswa (LKS) SMK dari berbagai tahun. Koleksi ini mencakup tingkat Kabupaten, Provinsi, hingga Nasional untuk membantu siswa dan pembimbing dalam mempersiapkan kompetisi teknis.

📌 Daftar Bidang Lomba
Arsip ini mencakup materi untuk bidang-bidang berikut:
IT Network System Administration (ITNSA)
Cloud ComputingWeb Technologies
IT Software Solution for Business (Aplikasi Perkantoran/Bisnis)
Graphic Design Technology

📁 Struktur Direktori
Folder dikelompokkan berdasarkan tahun pelaksanaan untuk memudahkan pencarian:
FolderDeskripsi2016 - 2025
Arsip soal dan kisi-kisi per tahun (Kabupaten/Provinsi/Nasional)2026
Update materi dan kisi-kisi terbaru untuk tahun berjalan
Soal-LKS-ITNSA-mainModul khusus dan konfigurasi lab untuk bidang ITNSA

🚀 Cara Menggunakan
Navigasi: Pilih folder sesuai tahun yang Anda cari.
Materi: 
Di dalam setiap folder tahun, terdapat sub-folder berdasarkan tingkat lomba atau bidang spesifik.
Unduh: 
Anda bisa melakukan git clone atau langsung mengunduh file PDF/Gambar yang tersedia di dalam direktori.
Bashgit clone https://github.com/ferdianrahabista/LKS-Kisi-Kisi-dan-Soal.git

🛠 Kontribusi
Jika Anda memiliki koleksi soal LKS yang belum tersedia di sini, silakan berkontribusi dengan cara:
Fork repositori ini.
Buat branch baru (git checkout -b tambah-soal-lks).Commit perubahan Anda (git commit -m 'Menambahkan soal Web Tech 2025').
Push ke branch tersebut (git push origin tambah-soal-lks).
Buat Pull Request.
